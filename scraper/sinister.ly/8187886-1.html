RE: MyBB Multi-color skin 08-12-2014, 09:48 AM #2
You can make your own method, you know? You really shouldn't try and replicate Sinisterly's method, since it's, to a certain extent, a glass cannon. Just learn JavaScript and the jQuery wrapper. Then combine some CSS, JavaScript, and a tad bit of HTML and you have a colour changing theme.

Here's five reasons why you SHOULDN'T use the JavaScript SL uses;
  • It is rather finicky and takes up much more space than it really should.
  • It has to change every image, and each colour needs its own directory and set of images.
  • The CSS is split up into too many files; honestly, a single file will do.
  • JavaScript noobs should beware; the JS isn't on a nooby level, and is much more advanced than you think, clearly. I'm not insulting you, but you're not all too good with JS. I've seen some themes you've done and a lot of the content in those themes are rips and mock ups.
  • It's for SL. It was custom coded. That means you'd be, in a sense, taking credit for someone else's work, and can bring up major copyright issues if anyone pulls a copyright claim against you. Whether it be Dynaxel, Audentio, or even Oni. All three played a major part in the creation of this multi-colour.

var yellowSwitch = document.getElementbyId("yellowSwitch");
var redSwitch = document.getElementById("redSwitch");

yellowSwitch.onclick = function() {
    //Elements to switch color of here:
    //An example of the <body> element changing color.
    document.body.style.backgroundColor = "yellow";
}

//add similar to above for other switches
