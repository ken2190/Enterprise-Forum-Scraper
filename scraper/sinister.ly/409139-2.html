RE: HC GUI's APP. 04-23-2012, 02:47 AM #13
I am willing to do this, I was thinking about this for a while and depending on what features you want this to have (and I have some suggestions of my own), I might need cooperation with staff, at least to install a plugin that I'll create for efficient push updates (instead of the client checking for new content all the time and thus using a lot of bandwidth, especially with many users, and thus requiring to decrease the check frequency (so you might be notified even after an hour for example), the forums itself pushes new updates to all active clients - or rather, it notifies them to check the updates right now, so you get instantly notified ) and other nifty features.

Additionally, my concern is that because new threads pop up very often, it might be quite annoying, so I would implement a way to filter it, so you'll be notified only of the threads that are interesting to you - you could filter by category, user category (for example ignore threads from newbies altogether) but also by tag in thread (text between [ ]) or any keyword or a phrase, including regular expressions for experienced users and that includes keywords in the title of the thread as well as the text itself (both configurable). Additionally, it might try to identify potential spam threads, which would make it useful for the staff as it would immediately point their attention to potentially problematic threads.

It might also inform you of new posts in threads you watch (+ the filters), PM's and other stuff if you want.

Regarding support, I can support following systems at first (all at once):
  • Windows
  • Linux
  • Mac OS X
Assuming there's interest, I can support also these platforms:
  • Android
  • iOS (iPhone, iPad, iPod Touch)
