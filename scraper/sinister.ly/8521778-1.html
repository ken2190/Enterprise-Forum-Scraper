How to protect your own MyBB forum! 01-25-2015, 09:31 AM #1
Hello and welcome to my first (maybe the 100th one, idk) tutorial. (pirate)

Today I will show you how to keep your forum safe from 1337 h4x0rz. It's pretty simple actually, but maybe not for the first time. But I will now provide some fixes for vulns, and security. So enjoy the tutorial guys, this tutorial is recommended if your board is not secured or you wanna know how. Let's start, shall we?

So the first thing we wanna do is change your Admin Control Panel (which is mainly /admin) to something more secure. To do so, you will have to follow the instructions below.

--

Step 1.

1. Logon to the FTP (or file manager), find the Admin directory.
2. Open up the folder and edit the "index.php" file inside.
3. Find
PHP Code:
$config['admin_dir'] = "admin";
} 

Now change "admin" to something more secure, I recommend using this site for STRONG characters: https://www.random.org/passwords/?num=1&len=16

After it's done, it should look something like this:

PHP Code:
$config['admin_dir'] = "emq355GWju4SVrf3y6WygVaYBBWaKR5evAju8UE69GbnUkzm";
} 

Once that's done, rename the Admin directory as above. The link you want to be the ACP basically (which is emq355GWju4SVrf3y6WygVaYBBWaKR5evAju8UE69GbnUkzm) in my case.

Now, head over to the "inc" directory and open the config.php file, and do the same pretty much. Find the code:

PHP Code:
$config['admin_dir'] = 'admin'; 

Change "admin" to the name your Admin directory is named after. Save it and now you're done. You have now successfully changed your Admin CP link!

--

Step 2. Now you have to protect your:
  • inc folder
  • acp/modules folder
  • acp/inc

Create a .htaccess file, now to the content, just write "deny from all" (without quotes) save it and upload it to the folders stated above.

Simple shit. Congratulations! now try to access the folder (thumbsup)

--

Step 3. Change your table prefix, well, how the fuck do I do that you might wonder? Well, with a simple .php script you are available to do so.

Download the file - http://www.mybbsecurity.net/attachment.php?aid=11

and upload it to the ROOT directory (/), once that's done, visit your board and do like this:

www.myboard.com/prefix_update.php (THIS SHOULD CHANGE YOUR PREFIX BY ITSELF), if you get a MySQL error, then you're fine. Now after this, remove the file.

Good thing, huh? You just protected your database from elite h4x0rzzz.

--

Step 4. Hide ACP

Make sure to hide it very well. If your account gets hacked, they can easily access the forum by pressing the "Admin CP" link in the welcome block, which IS not okay!

To do it, change

PHP Code:
$config['hide_admin_links'] 

It's a 0 there, right? change it to "1" (without quotes, of course).

--

Step 5. Here are some vulns (1.6 ONLY)

And how to fix!

First one:
www.myboard.com/search.php?action=results&sid[0]=9afaea732cb32f06fa34b1888bd237e2&sortby=&order=

How to fix:

Edit your "search.php" file and find the code
PHP Code:
$sid = $db->escape_string($mybb->input['sid']); 

change it to
PHP Code:
if(is_array($mybb->input['sid']))
    $sid = $db->escape_string(implode($mybb->input['sid']));
    else
    $sid = $db->escape_string($mybb->input['sid']);

Step 6.

If someone..did get access to the ACP, they could just easily download a backup of the forum. We don't want that! NO.

To disable the backups in acp, just do this:

Go to acp/modules/tools and edit the PHP file "backupdb.php"

Just right after the <?php tag, add

PHP Code:
die('Backups disabled, GET OUT OF HERE!'); 


--

Step 7.

Add a PIN to your admin panel, to do this you can follow this tutorial:

http://community.mybb.com/thread-105587.html

Step 8. Probably the most important part, ever..

KEEP YOUR FORUM UP TO DATE, AND USE UNIQUE PASSWORDS SUCH AS:

Code:

If you do all this, and the step above your forum will stay secure. This makes it the hacker have a hard time, so this is recommened to do if you want to start a forum.
