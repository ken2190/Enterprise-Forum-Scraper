Intersploit ~ hacking tools 05-19-2012, 11:26 AM #1
I've been working on a website that contains hacking tools of all sorts.
It's in a beta-like phase as there's only seven tools currently, however they're all done with precision and user friendliness in mind.
The website is also completely open-source, just click the Source tab to browser through the PHP source!
Suggestions, especially for tools, are taken in with pleasure. Read the About page before suggesting a tool though.

Current tools include:
  • Hash Identifier ~ This is powered by regular expressions which allow for more exact matching. All other hash type identifiers I've seen compare only length and whether or not it's alphanumerical.
  • Cookie Stealer ~ Generates a cookie stealer script based on variables given by the user.
  • Proxy Scraper ~ Scrapes the newest proxies off of HideMyAss with basic filtering options.
  • Shell Checker ~ Checks from a list of shells whether they're alive or not. You can pick from three different methods to check as well. Also, due to the project being open-source, you do not have to worry about security (e.g. the website logging your shell URLs).
  • Shell Collection ~ A list of downloadable shells. Upon download the password entered into the input field is automatically inserted into the shell's code. You can also download minified versions of all shells. In addition, all shells have been manually checked for backdoors.
  • String Tools ~ A handful of string tools useful for string manipulation, such as string length checking or base64 encoding. This is all done with Javascript and thus works instantly without page reloads.
  • Hash Cracker ~ Searches 25 hash cracking sites to retrieve the plaintext of a hash. Currently works only with MD5 hashes. Statistics about the site's response time is gathered and the fastest sites are searched through first.

I finished coding an admin page finder today, however I came to second thoughts and until further notice I will not release it.
NOTE: There was a problem with the shell collection where shells were corrupted and downloaded only up to 48 KB. This has been fixed.

Without further ado, I present to you Intersploit.
http://intersploit.net/

Also, I wasn't sure which section this belongs to, but due to it's close relativity to web-based hacking I decided to post it here.
