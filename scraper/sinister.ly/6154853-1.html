Sinisterly Chan Theme 06-18-2016, 06:18 PM #1
Hey all,

We've got a chan for SL, so I figured I'd share a pretty common theme I found that I use with 4chan (Twily's theme) because it works just as well with SL's chan.

I can't live without dark themes, and quite frankly I think the base Chan theme on all the chan sites is ugly af. so here.

(Don't mind the transparency, that's my desktop NOT this theme) ....yes that is Rias on the right

To use, install Stylish and "create new user style" Copy-Paste the code into the style along with your waifu (if you wish). You can comment it out with /* */



https://gist.github.com/InsidiousMind/7f...9f136a9832
