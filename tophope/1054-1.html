Тут приведен список самых популярных команд для ГМ.
//admin - админ-панель
Все что увидите ниже, будет доступно в самой админ-панели, которая названа выше.
Но, иногда удобней сделать макрос на данные команды.
//recall ник - телепорт игрока к себе
/teleport_to_character Телепортация к необходимому персонажу. Использование //teleport_to_character Ник
//res - оживить игрока, нужно навести таргет
//kill - убить игрока, таргет
//heal - вылечить игрока, таргет
//ban - ник\аккаунт, бан
//unban - ник\аккаунт, разбан
//kick - выбросить из игры персонажа, нужен таргет
//invul - Включить бессмертие.
//delete - При наведенном таргете удаляет любого моба или нпц из спавнлиста. Моб больше спавнится не будет.
//character_list - Открыть список игроков в онлайне.
//find_character - Открытие меню поиска персонажа по имени.
//gmshop - Открыть GMShop.
//itemcreate - Открывает меню создания итемов.
//create_item Создать определенный итем. Использование - //create_item item_id
//spawn Заспамнить определенного моба(нпц) в спавнлист. Использование: Встать в нужном месте, где хотите поставить нпц, взять себя в таргет и набрать команду - //spawn npc_id


Так же, есть такая вещь, как кнопки ALT+G. Как использовать?
1) Наводим таргет на игрока, которого нужно проверить
2) Жмем ALT+G
3) В появившемся окошке, проверяем правильно ли введен ник человека в таргете
4) Выбираем действие, например, посмотреть инвентарь - Inventory\Инвентарь
По аналогии с остальным тоже самое.

Дополняю Все, что вспомнил:

//setname - Изменяет имя персонажа (нужен таргет)
//setcolor - Изменяет цвет ника персонажа (формат RGB)
//setlevel - Изменяет уровень персонажа (таргет)
//settitle - Изменяет титул персонажа (таргет)
//teleport x y z - Телепортирует в указанные координаты.
//ride_название - Позволяет сесть на ездовое животное. Пример: //ride_wyvern. Обратная команда: //unride
//poly id - Позволяет превратить персонажа в любого НПЦ, сам персонаж не будет видеть трансформу, но окружающие будут. Обратно: //unpoly
//server_restart время(сек) - Позволяет перезагрузить сервер через определенное время, в секундах.
//announce текст - Объявление в чат голубым цветом.
//block - Позволяет обездвижить любого НПЦ или Персонажа, прикольно Обратно: //unblock
//enchant - Открывает меню заточки.
//chatban ник время(мин) причина - Позволяет забанить чат на определенное время, в минутах. Обратно: //chatunban ник
//invis - Включает невидимость. Обратно: //vis
//setclass ID - Позволяет дать профессию игроку. Пример: //setclass 102
//sethero - Позволяет стать героем (по таргету). Чтобы забрать героя: //sethero
//setnoble - Позволяет стать дворянином.
//gmspeed 1-4 - Позволяет включить режим супер ускорения. Наверно самая любимая команда... Пример: //gmspeed 4
//admin_reload - Открывает меню для перезагрузки компонентов сервера.
