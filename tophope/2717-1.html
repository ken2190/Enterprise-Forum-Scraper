Приветствую всех гостей и пользователей форума!
==================================================​

Данную программу можно использовать для восстановления доступа к потерянному дедику.
После ввода правильного пароля нам будут доступны следующие функции:

- Command Line (cmd.exe): Командная строка с правами SYSTEM
- Task Manager: Диспетчер задач с правами SYSTEM
- Local Users and Groups: Локальные пользователи и группы
- Computer Uptime: Точное время аптайма компьютера
- Clear Restore Points (Win 2003): Очистка всех точек восстановления для ОС Windows Server 2003 ( выполнять из Windows-a )
- Clear Restore Points (Win XP): Очистка всех точек восстановления для ОС Windows XP ( выполнять из Windows-a )
- Clear Restore Points (2008/Vista/Win7): Очистка всех точек восстановления для ОС Windows Server 2008, Windows Vista, Windows 7 ( выполнять из Windows-a )
- Clear Event Logs: Полная очистка логов с дедика (выполнять из winlogon-a (окна авторизации))

Для установки программы нужны права администратора.

Основные плюсы:
- запускается из winlogon-a в скрытом режиме ( читаем readme.txt )
- для активации/показа окна авторизации используется комбинация клавиш + пароль. Это дает гарантию что никто кроме вас не запустит программу.
- Полная автоматизация установки, не требует сторонних драйверов, dll-файлов, фреймворки.

Опционально:
* При покупке меняем комбинацию по вашему желанию, пароль сами криптуете. Если забыли пароль или комбинацию, это не наши проблемы.

Софт без привязки, то есть его можно использовать на неограниченное количество машин.

Все дальнейшие подробности о работе программы я объясняю только после приобретения софта. Все главные свойства вы уже прочитали.
* Личные советы и помощь вы получите в любое время, естественно когда я буду онлайн.
* Также дальнейшие обновления вы получите бесплатно, как только они будут разработаны.
* Готов рассмотреть любые ваши предложения и желания по улучшению функционала.
