С помощью данного урока я объясню, как можно нарисовать фон для меню или контента сайта в виде свитка пергамента.



Шаг 1. Рисуем заготовку.

Для начала рисуем форму нашего свитка. Я выбрал небольшую прямоугольную форму, т.к. на свитке у меня будет располагаться меню сайта.
Ставим прямоугольное выделение нужного вам размера и заливаем его необходимым цветом. Запомните, от цветов очень многое зависит, поэтому старайтесь использовать те оттенки, которые красиво впишутся в общую картину вашего дизайна.
Я выбрал в качестве основного цвета #d3cdbb.



Далее к слою с нарисованным прямоугольником применяем следующие стили:







В результате выполненных действий должен получиться подобный результат.



Помните, что тень Drop Shadow можно и не добавлять или же сделать её менее заметной или меньшего радиуса. Смотреть нужно по ситуации, старайтесь подстраивать параметры для достижения необходимого вам результата.

Шаг 2. Рисуем "рваные края".

С помощью инструмента Pen Tool (P) рисуем форму рваных краёв, примерно как показано на скриншоте. Старайтесь ставить как можно больше точек Pen Tool для достижения более красивой формы.



Затем соединяем Pen Tool по кругу, ставим выделение и заливаем его любым цветом, чтобы форму рваных краёв было хорошо заметно.



Форму с правой стороны можно не отрисовывать заново, просто продублируйте слой с формой и отобразите её по горизонтали и по вертикали.



Соединяем левую и правую формы, ставим выделение и вырезаем выделенную область у слоя с нашей заготовкой.



Понемногу начинает вырисовываться свиток




Шаг 3. Добавляем закрученные края.

Чтобы наша заготовка была ещё больше похожа на свиток пергамента, вверху и внизу нужно добавить закрученные части. Опять всё начинаем прорисовывать с простых геометрических фигур. Рисуем прямоугольник основного цвета #d3cdbb и располагаем его в верхней части свитка.



С помощью кругового выделения Eliptical Marquee Tool рисуем овал и заливаем его тёмным цветом #aa957f.





Теперь закрученной части нужно придать форму. Для этого нужно обрезать с помощью овального выделения верхний и нижний края.

Вырезаем выделенную область, затем проделываем такие же действия и для нижнего края.



Так же не забываем обрезать лишнюю часть слева.



Далее нужно придать объём для основной светлой части закрученного края нашего свитка.
Применяем к слою с основной частью стиль Gradient Overlay с данными настройками:



Ниже скриншот с параметрами самого градиента, чёрный цвет по центру смещён немного влево Location: 36%.



Должно получиться примерно так же.



Теперь нужно добавить внутреннюю закрученную часть свитка. Возвращаемся к слою, с ранее залитым тёмным овалом.



Так же как и для светлой основной части добавляем для данного слоя с тёмным овалом стиль Gradient Overlay. Настройки самого градиента, цвета: чёрный - белый (Location: 50%) - чёрный



Получается такой объёмный эффект.



Дублируем данный слой, трансформируем с соблюдением пропорций до 80%, убираем яркость Brightness / Contrast > Brightness: -40.
Перемещаем слой немного влево, примерно на 4-5px.



Теперь нужно добавить светлую часть внутренней закрученной части, чтобы у нас не было просто обычных 2 овала.
Дублируем слой со светлым овалом и ставим выделение ровно по высоте тёмного овала.



Теперь инвертируем выделение и вырезаем лишние части светлого продублированного овала.
Добавляем для данного слоя яркость Brightness: 20.



Объединяем все слои с закрученными краями и копируем их в правую сторону.



Для завершения обработки закрученных краёв необходимо добавить тень и светлый блик. Тень создаём с помощью чёрно-прозрачного градиента. Ставим выделения к основе свитка и рисуем сверху вниз 2 градиента для тени.
Первый градиент рисуем немного ниже и ставим прозрачность слоя на 8%.



Второй градиент немного выше и прозрачность на 11%.





Объединяем или группируем тени с закрученными краями, дублируем и располагаем все те же элементы внизу.



В завершении добавляем какую-нибудь текстуру и тени по краям. Ставим выделение по всей области свитка и заливаем текстуру.





Ставим для слоя с текстурой параметры Opacity: 40%, стиль наложения Soft Light.



Тени по краям рисуем с помощью круговых градиентов, с применением стиля наложения слоя Soft Light.





Как я уже отметил в начале урока, данные навыки можно применить для рисования больших фонов под контент сайта, к примеру для новостей.
Я же описал данный урок на основе создания фона для меню.
