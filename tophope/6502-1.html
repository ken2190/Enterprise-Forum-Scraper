VKClient - программа для рассылки и продвижения вконтакте. Рассылка сообщений, парсер групп и пользователей, автоответчик, первонах, чистка и заполнение страниц, менеджер аккаунтов для переписки

Рассылка вконтакте - очень действенный способ для продвижения своих товаров и услуг, раскрутки групп и привлечения новых клиентов. Программа VKClient содержит в себе весь необходимый для этого набор функций. Соберите целевые группы и участников сообществ с помощью парсера, выполните рассылку по стенам и комментариям групп, настройте первонах, чтобы вашу рекламу увидело максимальное число пользователей. Встроенный инвайтер поможет раскрутить группу с помощью рассылки приглашений вашим друзьям или друзьям ваших актив аккаунтов. Комментирование и лайкинг постов и аватарок пользователей просто не сможет не обратить внимание на вашу страницу. Чистите и заполняйте аккаунты, размещайте на стене свои офферы/сайты/партнерки, рассылайте личные сообщения или создавайте беседы, ставьте лайки пользователям, потом ведите с ними переписку прямо из программы и закрепляйте результат. Если вам лень переписываться, то настройте автоответчик и отправляйте собеседников на свои ресурсы напрямую. В боте есть максимальное число необходимых функций для добычи трафика вконтакте.

Рассылка
— Рассылка по личным сообщениям пользователей с возможностью использовать имя получателя
— Рассылка по беседам: создание, отправка, выход, удаление беседы у отправителя
— Рассылка по друзьям с критериями: онлайн, пол, возраст. Можно использовать имя получателя при обращении, заменяя его на уменьшительно-ласкательное по собственному словарю, что вызовет больше доверия (допускается использование нескольких вариантов для одного имени)
— Рассылка на стены групп
— Рассылка в личные сообщения групп
— Рассылка по альбомам групп (загрузка фото)
— Рассылка комментариев к постам и видеозаписям в группах
— Рассылка репостов по личным сообщениям
— Постинг на свою стену

Лайкинг
— Лайкинг аватарок и постов пользователей
— Лайкинг записей в группах
— Автолайкинг новых постов в группах
— Лайкинг по заданному списку постов

Репосты
— Автоматические репосты новых постов в группах
— Репостинг из групп
— Репосты по заданному списку постов
— Рассылка репостов по личным сообщениям

Комментарии
— Комментирование аватарок и постов пользователей
— Автоматическое комментирование новых постов в группах
— Комментирование заданного списка постов
— Функция первонаха (Оставление первого комментария под новым постом в группе)

Парсинг с фильтрацией
— Парсер групп из поиска
— Парсер пользователей из групп
— Парсер пользователей, лайкнувших запись
— Парсер друзей

Автоответчик и переписка
— Ведение диалогов прямо из программы с поддержкой вложений
— Автоответчик по ключевым словам и фильтром по полу
— Перепривязка сообщений с заблокированных аккаунтов

Заполнение аккаунтов
— Подписка на сообщества
— Добавление видео
— Добавление аудио
— Создание альбомов и загрузка в них фото
— Репосты из групп
— Изменение города
— Изменение статуса
— Изменение аватарки
— Изменение даты рождения
— Смена паролей

Чистка аккаунтов
— Чистка стены
— Чистка фото и альбомов

Инвайтинг, добавление в друзья, вступление в группы
— Инвайтер друзей в группы с критериями пол и возраст
— Рассылка заявок в друзья
— Добавление (прием заявок) в друзья
— Приветствие новых друзей
— Вступление в группы

Дополнительные возможности
— Удаление сообщений после отправки
— Добавление вложений при рассылке по стенам, комментариям и личным сообщениям
— Режим имитации активности человека: скроллинг новостей, переходы из ленты в паблики, на страницы друзей, рандомные лайки и репосты, лайки аватарок, написание сообщений. Отлично подойдет, если у вас есть авторег/ретрив аккаунты либо аккаунты, которыми давно не пользовались
— Поддержка прокcи, юзер агентов
— Учет дубликатов (блэк лист)
— Проверка ссылок на бан вконтакте
— Возможность начать работу по заданному времени
— Поддержка рандомизации текста
— Создание отчета о работе
— Во всех режимах есть имитация набора текста

Переписан автоответчик и мессенджер для ведения диалогов, расширена информация о диалоге
Автоответчик теперь работает в 2-х режимах

1. По списку ответов. В количестве от 1 до 5, с возможностью выбора числа ответов
— Можно указать номер автоответа, после которого прекратить отправку сообщений
— Поддержка рандомизации и вложений

2. Обучаемый автоответчик
— Тут вы задаете правила - ключевые слова или фразы, а также сообщения, которые будут отправлены, при наличии во входящем этих слов или фраз.
— Есть возможность указать номер сообщения, после которого отправить рекламу, можно продолжить общение после отправки рекламы или прекратить его
— Можно задать специальное правило, что отправлять, если ключей во входящем сообщении нет
— Поддержка рандомизации и вложений


— При создании альбома в режиме заполнения страниц больше не будет выдавать ошибку, если в названии есть пробел
— Исправлены рассылка репостов в лички и репосты из групп (не для режима по списку (авторепостинг), а при репостах в случайном порядке)
