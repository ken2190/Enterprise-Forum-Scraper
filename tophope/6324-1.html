Посмотреть вложение 656
Если у вас есть время протестируйте!​
Что добавил:
  1. Немного русификации
  2. При удаление чего либо, программа спросит вас удалить или оставить
  3. При клике выбора хроник сразу предложит открыть файл interface.xdat
  4. Добавил поиск и замена текста в скриптах
  5. Встроил java
  6. Добавил запуск программы на exe
  7. Теперь можно запустить игру прямо из редактора если у вас файл l2.exe
  8. Открыть копию программы из меню
  9. В файле description.csv можно добавить свое описание к названием полей
Тестируем отписываемся предлагаем свои варианты!)

Инструкция к применению):
  1. Распакуйте архив
  2. Зайдите в паку XDE и запустите файл XDE.exe

Возможно автор уже сделал и по другим , не смотрела )
Декомпилятор xdat подерживает только версию интерлюд
Как использовать
Для использование рекомендуется Java 1.8

Файл найстроек(compile.property):
xdat.original=target/interface.xdat
источник декомпиляции
xdat.src = target/xml/
папка декомпиляции, куда будут сохранены все файлы.
xdat.target = C:/Games/Interlude/system/interface.xdat
Назначение компиляции xdat файла из xml. Куда будет сохранен файл xdat.

Рекомендую указывать папку игры, и через ///reloadui перезапускать ui в игре, для динамической разработки.

В папке, есть два bat файла compile.bat и decompile.bat.
Один компилурет xml файлы. Другой декомпилирует xdat файл, в xml.
