Всегда начинайте рисование с простых геометрических фигур. Каждый элемент конечно же рисуйте на отдельном слое, это позволит вам в будущем работать с каждой частью отдельно.
Заливайте их лучше сначала серым цветом. Более светлым оттенком те части, которые будут находится ближе:



И более тёмным оттенком те, которые будут расположены немного дальше:



Так вам легче будет представить объём и расположение частей относительно друг друга.

Шаг 2. Придаём объём и рисуем тени.

Теперь будем работать с каждым элементом по отдельности, "оживляя" его с помощью передачи объёма и прорисовки теней.
Работать лучше всего с максимальным увеличением каждой части, т.к. на данном этапе работу нужно выполнять с точностью до 1-го пикселя.

Снова с помощью более тёмных оттенков серого цвета рисуем те части которые будут вдавлены и с помощью светлых, которые будут находится сверху:



Добавляем прямоугольный элемент. В процессе следующей обработки это будет цилиндрическая часть:



Теперь приступим непосредственно к самой обработке.

Затем выделяем внутреннюю часть получившейся чёрной рамки и удаляем её:



Выполняем фильтр Filter > Blur > Guassian Blur... c параметрами:



Снова устанавливаем для всех светлых частей выделение, выполняем меню Select > Inverse:



Удаляем лишние части Ctrl + x и устанавливаем для данного слоя прозрачность примерно на 70-80%:



Теперь работаем с тёмными частями. Им необходимо придать вдавленный эффект.
Для этого ставим выделение и с помощью линейного градиента (G) чёрного цвета рисуем градиент:



Для второй части совершаем те же действия, только градиент направляем в противоположную сторону:

Затем смещаем выделение на 2px вниз и вырезаем Ctrl + x:
