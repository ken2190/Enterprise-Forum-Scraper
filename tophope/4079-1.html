Шаг 1. Рисуем каменную основу.

Я думаю объяснять, как рисовать фон не стоит, тем более, что он может быть у вас свой, поэтому сразу начну с объяснения рисования каменной основы.
Для начала рисуем ориентир - примерную форму фона, который вы будете затем выкладывать камнями.



Ищем какую-нибудь картинку - реф с изображением камней. Желательно, чтобы разрешение было большое, иначе возникнут проблемы с нарезкой и итоговым качеством камней.
Я выбрал данную картинку, которую вы можете скачать в прикреплённых к теме файлах.



С помощью инструмента Pen Tool (P) делаем обводку нужного нам камня. Важно соединить концы линии Pen Tool, чтобы затем можно было сделать выделение по заданной кривой.



Выполняем меню Mouse2 > Make Selection со следующими параметрами:



Получается выделение нужной нам области камня:





Вырезаем по очереди каждый камень и вставляем его в свой исходник на отдельный слой, трансформируем и уменьшаем до нужного размера каждый камень, чтобы выложить необходимую нам форму каменной основы.



У меня получилась вот такая вот мозаика из камней.



Хочу отметить, что форму можно делать какой угодно, выкладывая при этом любой рельеф и изгибы.

Затем убираем все "больные" места. Это стыки камней, и белые пиксели по краям.



Сделать это легко, стыки убираем по принципу нарезки камней, при этом вырезая только часть ровного соединения камня. А белые пиксели и форму аккуратно затираем с помощью Eraser Tool (E).
Убираем слой с ориентиром формы - серый прямоугольник.



Теперь приступим к обработке.
Соединяем все камни в один слой (выбираем слои > Ctrl + E). Обесцвечиваем камни в том стучае, если вы выбрали цветные рефы камней, с помощью сочетания клавиш Ctrl + Shift + U.
Создаём под слоем с соединёнными камнями слой, который заливаем серым цветом (я выбрал цвет 626262), чтобы закрыть прозрачную область, которая образовалась в центре камней.



Теперь устанавливаем выделение по контуру камней и с помощью кругового градиента Gradien Tool рисуем плавный переход центрального цвета 626262.





Соединяем 3 слоя, с задней область, которая закрывает прозрачную середину камней, соединённые камни и верхний градиент с плавным переходом.

Корректируем яркость и контрастность так, чтобы настроенный тон камней соответствовал и совпадал с общей цветовой гаммой. Я выбрал следующие настройки:



Затем в стилях слоя устанавливаем Color Overlay, чтобы ещё немного притушить яркость камней и сделать цветовую гамму несколько мягче. Устанавливаем следующие значения:



Результат данных действий:



Выполняем меню Filter > Noise > Reduce Noise со следующими параметрами:



Создаём новый слой, ставим выделение по контуру и трансформируем его Select > Transform Selection до 99%, с соблюдением пропорций:



Трансформирование выделения до 99% позволит нам в будущем избежать появление светлых пикселей при объединении слоёв.

Заливаем данный слой любой повторяющейся текстурой.



Ставим стиль наложения слоя Overlay 20%:



И объединяем слои.

Затем нам предстоит добавить объём и глубину полученным камням. Для этого мы опять будем работать с инструментом Pen Tool, чтобы создать область выделения по кругу каменной основы.
Создаём новый слой поверх камней.
Ставим точечное выделение Pen Tool (P)



Соединяем его по кругу



Выполняем меню Mouse2 > Make Selection (0 pixels), а затем Select > Inverse, чтобы инвертировать выделение.



Далее с помощью кругового градиента в местах где вы хотите видеть светлые грани краёв камней рисуем градиенты



Теперь ставим выделение по контуру слоя с камнями и снова переходим на работу со слоем, где расположены белые градиенты, инвертируем выделение Select > Inverse. Вырезаем лишние части Ctrl + x.



Устанавливаем стиль наложения для слоя с градиентами Overlay 60%:



Создаём ещё один слой, ставим ему сразу стиль наложения Overlay 100% и с помощью кисти толщиной 1px наводим аккуратные грани на самых светлых частях камней.





Соединяем слои.

В настройках слоя выставляем следующие значения Inner Glow для создания рефлексов (отсветов):



Хочу отметить, что нужно выбирать тот цвет, который окружает ваш каменный элемент.

При возникновении слишком ярких видимых контуров, можно немного добавить прозрачность в настройках Inner Glow или затереть инструментом Eraser Tool края камней.

Добавляем немного резкости камням с помощью фильтра Filter > Sharpen > Unsharp Mask со значениями:



Для каменной основы я выставил параметры яркости -50, а для общего фона -30, чтобы сделать общую цветовую гамму темнее.





Ну вот, каменная основа для меню готова. Теперь приступим к следующему шагу рисования пунктов меню на дощечках.



Шаг 2. Рисуем деревянные дощечки с пунктами меню.

Рисование всех элементов интерфейса стоит начинать с простых геометрических фигур. Поэтому для прорисовки будущей дощечки я нарисовал прямоугольник.



Дублируем слой с прямоугольником, размещаем его ниже, делаем смещение вниз примерно на 5-6px и убираем Яркость (Brightness) на -30. Таким образом мы получаем нижний объём будущей дощечки:



Соединяем 2 слоя, ставим выделение по контуру и на новом слое делаем заливку деревянной текстурой.



Выставляем для данного слоя стиль наложения Overlay 50%:



Видим что текстура при наложении затемнила основу дощечки, поэтому к слою с основой ставим значение Яркости (Brightness) на +30:



Объединяем слои основы дощечки и верхней деревянной текстуры.
С помощью инструмента Pen Tool (P) рисуем в нижней части дощечки кривое выделение с различными трещинками.



Создаём выделение



И заливаем его линейным чёрным градиентом в направление сверху вниз.



Убираем лишние элементы градиента по контуру дощечки.



Устанавливаем прозрачность слоя с градиентом на 50%.



Соединяем слои. С помощь Pen Tool (P) создаём неровный контур дощечки.



Создаём выделение и инвертируем его Select > Inverse.



Убираем лишние части Ctrl + X.



C помощью кисти тёмно-коричневого цвета, наводим по бокам тёмные части дощечки



А с помощью светлого цвета кисти наводим грани и края



Применяем к слою с дощечкой стиль Color Overlay, c параметрами:



Это придаст дощечке более мягкую цветовую гамму.



Далее применяем фильтр Filter > Shapren > Unsharp Mask



Теперь добавим дощечке нижнюю тень. Для этого ставим выделение по контуру, создаём новый слой и заливаем его чёрным цветом.



Размещаем его под слоем с дощечкой и создаём дубликат данного слоя.

Шаг 3. Дополнения композиции.

Далее все зависит от вашей фантазии. Можно дорисовать травинки в трещинах, добавить тени, огни по бокам, наложить более атмосферный цветовой фильтр и т.д.
